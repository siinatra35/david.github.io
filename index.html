Projects

FullXTest

We wanted to create a web application which makes it easier for people to perform passive and active reconnaissance on any one target, and then to categorize each target on a vulnerability scale from 1 - 7 (low severity - high critical). The end result is a report as a summary on each host, compiled on the information gathered and if the attacks were successful or not. By exposing vulnerabilities, the host/system can be secured and further protected.

CCNA-project-in-packet-tracer

This project is a simple simulation of a network topology that is similarly used in a college. This topology implements the following protocols DHCP, OSPF, EIGRP, VLANs, and NAT. And implements specified VLAN groupings. OSPF and EIRGP connections configurations. And demonstrates ip remapping using NAT.

CTF & Hackthebox write-ups

This project is actually a write-up library covering CTFs and Hackthebox. For those that are not familiar with capture the flags, it is a cybersecurity competition where participants solve computer security problems or capture and defend computer systems. And Hackthebox is a platform where you can practice pentesting and solve challenges. And write-ups are detailed documentation of the step by step process that was used to solve a problem or machine.
